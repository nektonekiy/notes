# `Promise`  
`Promise` – это специальный объект, который содержит своё состояние. Вначале `pending` («ожидание»), затем – одно из: `fulfilled` («выполнено успешно») или `rejected` («выполнено с ошибкой»).  
На promise можно навешивать `callback`\-и двух типов:  
`onFulfilled` – срабатывают, когда promise в состоянии «выполнен успешно».  
`onRejected` – срабатывают, когда promise в состоянии «выполнен с ошибкой»..

```js
// Сначала мы создаём Promise, конструктор которого принимает в себя функцию, в нее мы передаём параметры, в который конструктор как аргументы передаст функции для выброса состояния
let examplePromise = new Promise(function(resolve, reject)) { 
// Далее тут мы можем выполнить любой нужный нам код,
// И если мы хотим выбросить успешное окончание или ошибку, 
// то мы вызываем соответствующие функции
// Если мы хотим вернуть какое то значение, то передаем её в функцию
if (somethingBadHappend)
	reject(error); // а это ошибочка 
resolve(data); *// это успешное выполнение*};
```
 Чтобы обработать `Promise` мы можем использовать следующие методы:

`.then(onFulfilled, onRejected)` \- в данный метод, мы можем передать функции которые будут выполнены при успешном выполнении `onFulfilled` и ошибке `onRejected`.

 Если в какой-либо параметр не будет добавлена функция, то это не проблема.

 Также функция может принимать данные с Promise.

 Обычно `onRejected` в этом методе не используется, так как есть метод `.catch(onRejected)`  
 `.catch(onRejected)` \- аналог `.then`, только для ошибок.  
 `.finally(onFinish)` \- после выполнения выполниться переданная функция.
