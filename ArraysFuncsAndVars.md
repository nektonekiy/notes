# Методы и значения у массивов

## Длина массива

`Array.length` - длина массива

## Изменение массива

`Array.push(<элемент>)` - добавить элемент в конец массива `Array.unshift(<элемент>)` - добавить элемент в начало массива

---

`Array.pop()`- удалить элемент из конца массива Array.shift()- удалить элемент из начала массива

---

`Array.concat(<другой массив>)` - вернёт массив объединенный с другим массивом

## Определение индекса

`Array.indexOf(<элемент>)`- определение индекса у элемента

## Превращение в строку

`Array.join(<разделитель>)` - превратит массив в строку с элемента массива разделёнными разделителем, который по умолчанию является ","
